##IDEA中文件名不同颜色的意思  

**绿色**，已经加入版本控制暂未提交；   
**红色**，未加入版本控制；   
**蓝色**，加入版本控制，已提交，有改动；    
**白色**，加入版本控制，已提交，无改动；   
**灰色**：版本控制已忽略文件。

##String转Boolean
>如果字符串参数不为 null 且在忽略大小写时等于 "true"，则返回的 Boolean 表示 true 值。其他均返回false    
                    
                            String a = "false";      
                            Boolean.valueOf("");
